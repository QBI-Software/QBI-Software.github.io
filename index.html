QBI-Software

Custom Software Services and Resources for QBI, The University of Queensland

Welcome to the QBI Software Site

A recent addition to QBI services is the capability to develop and manage custom software applications, macros, plugins, scripts or just provide assistance and advice on software matters for all staff and students at QBI free of charge.

Applications

Custom software applications can be developed for QBI research teams to assist with analysis or data management. Examples so far include applications for neurite analysis in Matlab, particle tracking in Python and a frog database in Django.

Scripts, Macros and Plugins

Data and image analysis often requires several steps in preparing the data or image, extraction of features and finally, production of output data for further analysis. , as well as NEURON models Automation into a bulk operation can drastically reduce analysis time from weeks to minutes. Maybe you need full custom development or just assistance with plugins, macros or scripts in Python, Matlab or R. Integration with Fiji/ImageJ, Matlab or other open source software packages is also supported as is parsing Excel or CSV data in Python or R.

Training

Introductory training courses are available covering basic computing skills such as using the Unix shell as well as user-friendly training for Python and R. Requests for training in specific languages or technologies are welcome.

Support or Contact

Useful code can be kept in versioned repositories on the public QBI-Software github repository so others may benefit. If you have a good script or code package you'd like to share please contact me below to be added as a contributor. Code access is available to all via the "Download" buttons available within each project. For further information, please contact Liz Cooper-Williams, Research Software Developer, QBI, e.cooperwilliams@uq.edu.au.
