QBI Software

Welcome to the QBI Software site!

A recent addition to QBI services is the capability to develop and manage custom software applications, macros, plugins, scripts or just provide assistance and advice on software matters.

Applications

Custom software applications can be developed for QBI research teams to assist with analysis or data management.

Examples so far include applications in Matlab, Python and Django, as well as NEURON models.

Scripts

Repetitive analysis of data can often be automated using macros or scripts, especially for image analysis in Fiji/ImageJ or Excel sheet data in python or R.

Useful scripts are added to the QBI-Software github repository so others may benefit. If you have a good script you'd like to share please contact me below.

Training

Introductory training courses will be available for all staff and students at QBI covering basic computing skills such as using the Unix shell as well as user-friendly training for Python and R.

Requests for training in specific languages or technologies are welcome.

For most projects, the code is kept in versioned repositories on Github for public access, available for sharing and contributing. For further information, please contact Liz Cooper-Williams, Research Software Developer, QBI (e.cooperwilliams@uq.edu.au)
